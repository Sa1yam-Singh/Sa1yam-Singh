| Security    | Secure Wireless Communication                                                                        |
| Impact      | Reduced Downtime & Faster Maintenance                                                                |
| Repository  | [View Repository](https://github.com/Sa1yam-Singh/Real-Time-Low-Voltage-Line-Fault-Detection-System) |

### Highlights

* RMS current measurement

* Expandable node infrastructure
